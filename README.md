# monorepo-lerna
# Usage

Install lerna first:
```bash
npm install --global lerna
```

Run ``` npx lerna bootstrap --hoist ``` to install all dependencies and setup monorepo symlinks using lerna.

Run ``` npm run start ``` to start development server
